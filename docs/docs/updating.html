Updating

Keep your LazyPi packages current.

Update installed packages

npx @robzolkos/lazypi update

This runs Pi's updater for installed package changes.

What the update does

When you run update, LazyPi:

  1. Refreshes Compound Engineering when it is installed
  2. Runs pi update for installed Pi packages
  3. Reports what changed

Update preserves your selected catalog footprint.
